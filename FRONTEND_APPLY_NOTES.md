# Как применять

Это готовые исходники с внесёнными правками. Для backend на проде не заменяй файл `app.db`: в архиве он специально не приложен, чтобы не затереть существующую БД.

Backend:
1. Остановить сервис.
2. Сделать бэкап текущей папки и БД.
3. Заменить файлы проекта из архива, оставив текущий `app.db` и `.env`.
4. Запустить backend обычной командой `uvicorn main:app --host 0.0.0.0 --port 8000`.

Frontend:
1. Выполнить `npm install`, если `node_modules` нет или он старый.
2. Выполнить `npm run build`.
3. Задеплоить новую сборку.

Проверено: backend Python-файлы проходят `py_compile`, frontend проходит `tsc -b`. Vite build в этой среде не был завершён из-за отсутствующей optional-зависимости Rollup в приложенном `node_modules`; после `npm install` должен собираться штатно.
